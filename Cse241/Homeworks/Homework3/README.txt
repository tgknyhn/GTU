Welcome to Homework 3
---------------------

To compile the game just write "make".

If it says: "file is up to date", then first enter "make clean" 
after that enter "make" to execute the game.

In menu section you have 6 choice.


1) NEW GAME
-----------

- When a new game starts you will choose board size and an opponent.

- To make a move first enter a column then a row (ex: A 11) !space is important

- If game ends you will be sent to Menu.

- If you want to load a game with existing .txt file you can simply write "LOAD filename.txt"
and the current game will be replaced with given .txt game.(not recommended since you can select
existing games in menu section)

- If you want to go to the menu before finishing game simply write "MENU 1", 

- Your game will be saved if you go to menu with or without finishing it.


2) LOAD GAME
------------

- All games will be listed here. 

- There are already 5 games going on when you execute the game. 

- All information you'll need before game selection, it will be shown to you 
(boardsize, opponent, moves played)

- To continue a game just write the number of the game you want to continue

3) Comperision
--------------

- All games will be listed here as well

- To compare enter two numbers which represents their game number. 

- Example input: "3 5" (without brackets) ! space is important

4) Move Count
-------------

- Shows sum of move counts of every game.

5) Print Board
-------------

- You can just see the board`s of the games without playing them. 

6) Exit
-------

- Game will continue as long as you don't give input as: "6" (without brackets)
